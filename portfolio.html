About me

I'm a software engineer with a focus on embedded systems and computer vision. My work sits at the intersection of hardware and software — building systems that interact with the real world, detect patterns, and respond in real time.

I've built driver safety systems using both Arduino microcontrollers and Python-based computer vision pipelines, combining sensor data, facial recognition, voice alerts, and mobile notifications to solve a genuinely life-critical problem. The system also includes database support via MongoDB for logging and tracking drowsiness events.

I enjoy working close to the hardware — writing efficient embedded C, wiring up circuits, and tuning OpenCV pipelines frame by frame. I'm driven by projects that have a clear real-world impact.
